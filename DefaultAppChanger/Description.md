# Default App Changer
## Requirments
AutoHotkey 2.0

Windows 10 only
## Description
Changes the default browser to Microsoft Edge, the default media player to Windows Media Player Legacy, and the default video player also to Windows Media Player Legacy
## What makes me create this file and put it on GitHub?
It reminded me when I was on a VM and installed AVG Secure Browser, yep.

It randomly went into the settings app and tried to **change default app settings**...I **stopped** that.
